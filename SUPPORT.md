## 常见问题

### Q: 如何在本地运行文档？

```bash
git clone https://github.com/Digidai/gtm-cookbook.git
cd gtm-cookbook
npm install
npm run dev
```

### Q: 如何贡献内容？

请阅读 [CONTRIBUTING.md](CONTRIBUTING.md) 了解贡献流程。
