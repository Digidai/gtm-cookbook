# 模块四：GTM 实战案例深度解析

> **学习目标**：通过经典案例学习 GTM 战略的实际应用
>
> **预计时长**：2 小时
>
> **教学方式**：案例研讨

---

## 模块概述

理论需要案例来落地。本模块选取了不同 GTM Motion、不同市场阶段的经典案例，深度解析它们的 GTM 战略选择、执行路径和成功要素。

通过本模块的学习，你将能够：
- 理解 PLG 公司如何从0到1增长
- 学习 SLG 公司如何攻占企业市场
- 掌握跨越鸿沟的实战经验
- 了解 AI 时代 GTM 的新趋势

## 案例学习方法

每个案例将从以下维度进行解析：

1. **背景**：公司背景、市场环境
2. **GTM 战略选择**：Motion、渠道、定位
3. **关键执行要素**：增长引擎、核心策略
4. **数据与结果**：关键指标、增长轨迹
5. **GTM Lesson**：可复用的经验教训

---

## 学习建议

1. 带着问题阅读案例
2. 对比不同案例的异同
3. 思考如何应用到自己的场景
4. 案例讨论时积极参与
